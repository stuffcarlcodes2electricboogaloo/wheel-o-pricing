# Wheel-o-Pricing

Wheel-o-Pricing is a responsive browser game that spins a pricing wheel and lands on a randomized value from a selected pricing category.

## Pricing Categories

- **Corporate:** $11,000 to $200,000
- **Enterprise:** $200,001 to $500,000
- **Strategic:** $500,001 to $1,000,000

## Run Locally

Since this is a static web app, open `index.html` in a browser.

For a local server (recommended):

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Optional Spin Theme Audio

You can upload your own licensed audio file in the app. If loaded, it loops while the wheel spins and stops when the spin ends.
